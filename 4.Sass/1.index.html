Entendendo o Sass

Coisas que o sass oferece a mais:

  • Variáveis: ajuda a acessar cores, por exemplo, com mais exatidão e mais fácil;
  • Agrupamento: ao invés de ter que estilizar tags de forma separada é possível agruar a tag pai com as tags filhos (colocando o & é possível colocar ações junta dos elementos);
  • Importar: é possível importar outros códigos do scss;
  • Mixin: como se fosse uma função do css
  • Extend: extende as configurações de um seletor para o outro sem precisar copiar e colar todas.
