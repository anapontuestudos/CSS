  • A propriedade display permite que você defina a maneira como determinado elemento HTML deve ser renderizado;
  • 'h1' ou 'p', são tags que tem por padrão o display block que ocupa a tela inteira na (horizontal);
  • Já tags como 'a' ou 'span' tem um display inline, que só ocupa uma parte da linha que está inserido;
  • display-none: esconde o conteúdo (junto com o javascript é possível fazer coisas interessantes);
  • display: inline-block: une as possibilidades dessas propriedades
