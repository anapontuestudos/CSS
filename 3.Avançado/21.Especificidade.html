Entendendo o CSS

  1. Ter uma estilização inline no seu código é hierarquicamente mais importante que ter um arquivo css fora;
  2. Ids são mais "fortes" do que classes;
  3. Especificar o elemento e a classe do elemento da mais prioridade;
  4. Classes tem mais prioridade em relação aos seletores simples;
  5. Se houverem muitos seletores simples apontando para um único elemento o navegador dará prioridade para o último
